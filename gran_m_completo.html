Método de la Gran M - Análisis Completo con Variables Auxiliares

Modelo Matemático Original

Minimizar Z = 2x1 + 3x2

Sujeto a:

  • 1x1 + 1x2 >= 4
  • 2x1 + 1x2 >= 6
  • x1, x2 ≥ 0

Modelo Extendido con Variables de Holgura y Artificiales

Análisis de Restricciones:

  • Restricción 1: 1x1 + 1x2 >= 4
        → Se agrega variable de exceso H1 ≥ 0
        → Se agrega variable artificial a1 ≥ 0
  • Restricción 2: 2x1 + 1x2 >= 6
        → Se agrega variable de exceso H2 ≥ 0
        → Se agrega variable artificial a2 ≥ 0

Modelo Extendido Completo:

Minimizar Z = 2x1 + 3x2 + 0H1 + 0H2 + Ma1 + Ma2

Sujeto a:

  • 1x1 + 1x2 - H1 + a1 = 4
  • 2x1 + 1x2 - H2 + a2 = 6
  • x1, x2, H1, H2, a1, a2 ≥ 0

Pasos para Construir la Tabla Inicial (Iteración 0)

Paso 1: Identificación de Variables

Variables originales: x1, x2, ..., x2

Variables de exceso: H1, H2, ..., H2

Variables artificiales: a1, a2, ..., a2

Total de variables: 6

Paso 2: Construcción de la Matriz de Coeficientes

La matriz se construye columna por columna:

  • Variables originales (x1, x2, ...): Se copian directamente los coeficientes de las restricciones
  • Variables de exceso (Hi): Coeficiente -1 en su restricción correspondiente, 0 en las demás
  • Variables artificiales (ai): Coeficiente +1 en su restricción correspondiente, 0 en las demás

Paso 3: Identificación de la Base Inicial

La base inicial está formada por las variables que tienen coeficiente 1 en exactamente una restricción y 0 en las demás:

  • Restricción 1: Variable básica a1
  • Restricción 2: Variable básica a2

Base inicial: {a1, a2}

Paso 4: Construcción de la Fila Z

La fila Z se construye de la siguiente manera:

  1. Variables originales: Se copian los coeficientes de la función objetivo
  2. Variables de holgura y exceso: Coeficiente 0
  3. Variables artificiales: Coeficiente M (para minimización) o -M (para maximización)
  4. Término independiente: Inicialmente 0

Paso 5: Eliminación de Variables Artificiales de la Fila Z

Como las variables artificiales están en la base inicial con coeficiente M ≠ 0 en la fila Z, debemos eliminarlas usando operaciones elementales:

Para cada variable artificial ai en la base:

  1. Identificar la fila donde ai es variable básica (coeficiente = 1)
  2. Multiplicar esa fila por -M (o +M según el caso)
  3. Sumar el resultado a la fila Z

Esto garantiza que las variables artificiales tengan coeficiente 0 en la fila Z.
