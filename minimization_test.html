Método de la Gran M - Informe Completo

Modelo Matemático Original

Minimizar Z = 2x1 + 3x2

Sujeto a:

  • 1x1 + 1x2 >= 4
  • 2x1 + 1x2 >= 6
  • x1, x2 ≥ 0

Modelo Matemático Transformado (con variables auxiliares)

Minimizar Z = 2x1 + 3x2 + Ma1 + Ma2

Sujeto a:

  • 1x1 + 1x2 - e1 + a1 = 4
  • 2x1 + 1x2 - e2 + a2 = 6
  • x1, x2, e1, e2, a1, a2 ≥ 0

Donde:

  • Variables originales: x1, x2
  • Variables de exceso: e1, e2 (para restricciones ≥)
  • Variables artificiales: a1, a2 (para iniciar el método simplex)
  • M: Número muy grande (M = 1,000,000) para penalizar variables artificiales
